# EchoHeli-TC1030\_Proyecto\_Integrador

This project is going to be focused on simulating a part of the game Super Smash Bros. Melee using classes in C++ to integrate several of the characteristics that fighters in the game have using a class structure with inheritance. The goal of the project is for characters to be able to interact with each other based on user inputs, with each character having certain movesets and attacks, the program will be able to detect which scenarios will the moves be performed, the effect of the moves, and the project also includes overwriting to fit the fighter's unique moveset.



The UML includes some cases where the project could stop working, so I will have to keep in mind other cases where this could happen and adjust my code accordingly
